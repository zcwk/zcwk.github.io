### Dockerfile的编写

``` java
FROM hub.c.163.com/library/java:8-alpine

MAINTAINER ade zdzzyc@gmail.com

ADD target/*.jar app.jar

EXPOSE 80

ENTRYPOINT ["java","-jar","/app.jar"]
```

### 构建docker镜像

docker build -t zdzzyc/manager .

### 运行docker镜像

docker run -p 8762:80 -d zdzzyc/manager
